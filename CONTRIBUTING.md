## How to Contribute / 如何贡献

### 🇺🇸 English

1. **Fork the repository** - Click the "Fork" button in the top right
2. **Clone your fork** - `git clone https://github.com/YOUR_USERNAME/mu-code.git`
3. **Create a branch** - `git checkout -b feature/your-feature-name`
4. **Make your changes** - Write clean, well-documented code
5. **Test your changes** - Run `npm run dev` to test
6. **Commit your changes** - `git commit -m "Add your feature"`
7. **Push to your fork** - `git push origin feature/your-feature-name`
8. **Open a Pull Request** - Go to GitHub and click "New Pull Request"

### 🇨🇳 中文

1. **Fork 仓库** - 点击右上角的 "Fork" 按钮
2. **克隆你的 Fork** - `git clone https://github.com/YOUR_USERNAME/mu-code.git`
3. **创建分支** - `git checkout -b feature/your-feature-name`
4. **进行修改** - 编写干净、文档完善的代码
5. **测试修改** - 运行 `npm run dev` 测试
6. **提交修改** - `git commit -m "Add your feature"`
7. **推送到 Fork** - `git push origin feature/your-feature-name`
8. **打开 Pull Request** - 进入 GitHub 点击 "New Pull Request"

## Development Setup / 开发环境设置

```bash
# Install dependencies / 安装依赖
npm install

# Start development server / 启动开发服务器
npm start

# Build for production / 生产环境构建
npm run build
```

## Code Style / 代码风格

- Use TypeScript for type safety / 使用 TypeScript 保证类型安全
- Follow existing code patterns / 遵循现有代码模式
- Write meaningful commit messages / 编写有意义的提交信息
- Add comments for complex logic / 为复杂逻辑添加注释
